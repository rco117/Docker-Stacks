# Docker Compose Stacks

Dieses Repository enthält verschiedene Docker-Compose-Stacks aus dem lokalen Docker-Ordner.

## Enthaltene Stacks

- Bentopdf
- OhMyPCAP
- Stirling PDF
- Termix
- Zerobyte

## Hinweise

- Die Compose-Dateien sind für die lokale Nutzung gedacht.
- Konfigurationsdateien und lokale Datenverzeichnisse sollten je nach Container angepasst werden.
- Sensitive Werte wie Passwörter oder Tokens sollten vor dem Upload ersetzt oder über Umgebungsvariablen verwaltet werden.
